连续型隐变量模型与变分自编码器

Sources:

  • Notebook
  • Repository

第 1 部分 本系列文章介绍了具有离散隐变量的隐变量模型、高斯混合模型 (GMM) 和拟合算法这个模型要数据，EM 算法。第 2 部分介绍了具有连续隐变量的隐变量模型，用于对更复杂的数据（例如自然图像）进行建模，以及可与随机优化算法结合使用的贝叶斯推理技术。

Consider a natural image of size 100×100100 \times 100100×100 with a single channel. This image is a point in 10.00010.00010.000-dimensional space. Natural images are usually not uniformly distributed in this space but reside on a much lower-dimensional manifold within this high-dimensional space. The lower dimensionality of the manifold is related to the limited degrees of freedom in these images e.g. only a limited number of pixel value combinations are actually perceived as natural images.

考虑一个大小为 100×100100 \times 100100×100 的自然图像，带有单个通道。此图像是 100001000010000 维空间中的一个点。自然图像通常不是均匀分布在这个空间中，而是驻留在这个高维空间内的一个低得多的流形上。流形的较低维度与这些图像中有限的自由度有关，例如只有有限数量的像素值组合实际上被视为自然图像。

Modeling natural images with latent variable models whose continuous latent variables represent locations on the manifold can be a useful approach that is also discussed here. As in part 1, a model with one latent variable ti\mathbf{t}_iti​ per observation xi\mathbf{x}_ixi​ is used but now the latent variables are continuous rather than discrete variables. Therefore, summations over latent variable states are now replaced by integrals and these are often intractable for more complex models.

使用潜在变量模型对自然图像进行建模，其连续的潜在变量表示流形上的位置，这也是一种有用的方法，这里也将讨论。与第 1 部分一样，使用每个观察值具有一个潜在变量 ti\mathbf{t}_iti​ 的模型 xi\mathbf{x}_ixi​ 但现在潜在变量是连续变量而不是离散变量。因此，对潜在变量状态的求和现在被积分所取代，对于更复杂的模型来说，这些通常是难以处理的。

Observations i.e. images X=x1,…,xN\mathbf{X} = \\{ \mathbf{x}_1, \ldots, \mathbf{x}_N \\}X=x1​,…,xN​ are again described with a probabilistic model p(x∣θ)p(\mathbf{x} \lvert \boldsymbol{\theta})p(x∣θ). Goal is to maximize the data likelihood p(X∣θ)p(\mathbf{X} \lvert \boldsymbol{\theta})p(X∣θ) w.r.t. θ\boldsymbol{\theta}θ and to obtain approximate posterior distributions over continuous latent variables. The joint distribution over an observed variable x\mathbf{x}x and a latent variable t\mathbf{t}t is defined as the product of the conditional distribution over x\mathbf{x}x given t\mathbf{t}t and the prior distribution over t\mathbf{t}t.

观测（ 即图像 X={x1,…,xN}\mathbf{X} = \{ \mathbf{x}_1, \ldots, \mathbf{x}_N \}X={x1​,…,xN​} ） 再次用概率模型 p(x∣θ)p(\mathbf{x} \lvert \boldsymbol{\theta})p(x∣θ)。目标是最大化数据似然 p(X∣θ)p(\mathbf{X} \lvert \boldsymbol{\theta})p(X∣θ) w.r.t. θ\boldsymbol{\theta}θ 并获得连续潜在变量的近似后验分布。观察变量 x\mathbf{x}x 和潜在变量 t\mathbf{t}t 上的联合分布被定义为给定 t\mathbf{t}t 上 x\mathbf{x}x 的条件分布的乘积以及 t\mathbf{t}t 上的先验分布。

p(x,t∣θ)=p(x∣t,θ)p(t∣θ)(1)p(\mathbf{x}, \mathbf{t} \lvert \boldsymbol{\theta}) = p(\mathbf{x} \lvert \mathbf{t}, \boldsymbol{\theta}) p(\mathbf{t} \lvert \boldsymbol{\theta}) \tag{1} p(x,t∣θ)=p(x∣t,θ)p(t∣θ)(1)

We obtain the marginal distribution over x by integrating over t.

p(x∣θ)=∫p(x∣t,θ)p(t∣θ)dt(2)p(\mathbf{x} \lvert \boldsymbol{\theta}) = \int p(\mathbf{x} \lvert \mathbf{t}, \boldsymbol{\theta}) p(\mathbf{t} \lvert \boldsymbol{\theta}) d\mathbf{t} \tag{2} p(x∣θ)=∫p(x∣t,θ)p(t∣θ)dt(2)

This integral is usually intractable for even moderately complex conditional probabilities p(x∣t,θ)p(\mathbf{x} \lvert \mathbf{t}, \boldsymbol{\theta})p(x∣t,θ) and consequently also the true posterior.

p(t∣x,θ)=p(x∣t,θ)p(t∣θ)p(x∣θ)(3)p(\mathbf{t} \lvert \mathbf{x}, \boldsymbol{\theta}) = {p(\mathbf{x} \lvert \mathbf{t}, \boldsymbol{\theta}) p(\mathbf{t} \lvert \boldsymbol{\theta}) \over p(\mathbf{x} \lvert \boldsymbol{\theta})} \tag{3} p(t∣x,θ)=p(x∣θ)p(x∣t,θ)p(t∣θ)​(3)

L(θ,q)=log⁡p(X∣θ)−KL(q(T∣X)∣∣p(T∣X,θ))(4)\mathcal{L}(\boldsymbol{\theta}, q) = \log p(\mathbf{X} \lvert \boldsymbol{\theta}) - \mathrm{KL}(q(\mathbf{T} \lvert \mathbf{X}) \mid\mid p(\mathbf{T} \lvert \mathbf{X}, \boldsymbol{\theta})) \tag{4} L(θ,q)=logp(X∣θ)−KL(q(T∣X)∣∣p(T∣X,θ))(4)

In the E-step, the lower bound is maximized w.r.t. qqq and θ\boldsymbol{\theta}θ is held fixed. If the true posterior is tractable, we can set qqq to the true posterior so that the KL divergence becomes 000 which maximizes the lower bound for the current value of θ\boldsymbol{\theta}θ. If the true posterior is intractable approximations must be used.
在 E-step 中，下界被最大化 w.r.t. qqq 和 θ\boldsymbol{\theta}θ 保持固定。如果真实后验是可处理的，我们可以将 qqq 设置为真实后验，以便 KL 散度变为 000，从而最大化 θ\boldsymbol{\theta}θ 当前值的下限。如果真正的后验是棘手的，则必须使用近似值。

Here, we will use stochastic variational inference, a Bayesian inference method that also scales to large datasets[1]. Numerous other approximate inference approaches exist but these are not discussed here to keep the article focused.
在这里，我们将使用随机变分推理，这是一种贝叶斯推理方法，也适用于大型数据集[1]。存在许多其他近似推理方法，但为了保持本文的重点，这里不讨论这些方法。

In this context, qqq is called a variational distribution and L(θ,q)\mathcal{L}(\boldsymbol{\theta}, q)L(θ,q) a variational lower bound.

涵盖函数 w.r.t. 优化的数学领域。在我们的例子中，像 argmaxqL(θ,q){\mathrm{argmax}}_q \mathcal{L}(\boldsymbol{\theta}, q)argmaxq​L(θ,q) 这样的函数是[变体演算](https://en.wikipedia.org /wiki/Calculus_of_variations)，因此得名变分推理。在这种情况下，qqq 被称为变分分布，L(θ,q)\mathcal{L}(\boldsymbol{\theta}, q)L(θ,q) 被称为变分下界。

Mini-batches

小批量

Variational autoencoder

变分自编码器

From the perspective of a generative model, q(ti∣xi,ϕ)q(\mathbf{t}_i \lvert \mathbf{x}_i, \boldsymbol{\phi})q(ti​∣xi​,ϕ) is a probabilistic encoder because it generates a latent code ti\mathbf{t}_iti​ for input image xi\mathbf{x}_ixi​ and p(xi∣ti,θ)p(\mathbf{x}_i \lvert \mathbf{t}_i, \boldsymbol{\theta})p(xi​∣ti​,θ) is a probabilistic decoder because it generates or reconstructs an image xi\mathbf{x}_ixi​ from latent code ti\mathbf{t}_iti​. Optimizing the variational lower bound w.r.t. parameters θ\boldsymbol{\theta}θ and ϕ\boldsymbol{\phi}ϕ can therefore be regarded as training a probabilistic autoencoder or variational autoencoder (VAE)[1].

In this context, the first term on the RHS of Eq. (5)(5)(5) can be interpreted as expected negative reconstruction error. The second term is a regularization term that encourages the variational distribution to be close to the prior over latent variables. If the regularization term is omitted, the variational distribution would collapse to a delta function and the variational autoencoder would degenerate to a “usual” deterministic autoencoder.

These definitions are used to implement a VariationalAutoencoder model class.

The variational_lower_bound function is implemented using Eq. (12)(12)(12) and Eq. (11)(11)(11) but instead of estimating the lower bound for the full dataset it is normalized by the dataset size NNN.

The training procedure uses the negative value of the variational lower bound as loss to compute stochastic gradient estimates. These are used by the optimizer to update model parameters θ\boldsymbol\thetaθ and ϕ\boldsymbol\phiϕ. The normalized variational lower bound of the test set is computed at the end of each epoch and printed.

Since the data are modelled with a multivariate Bernoulli distribution, the MNIST images are first binarized to monochrome images so that their pixel values are either 0 or 1. The training batch size is set to 100 to get reliable stochastic gradient estimates.

We choose a two-dimensional latent space so that it can be easily visualized. Training the variational autoencoder with RMSProp as optimizer at a learning rate of 1e-3 for 20 epochs gives already reasonable results. This takes a few minutes on a single GPU.
