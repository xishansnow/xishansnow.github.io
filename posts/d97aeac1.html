1️⃣ 有向概率图模型概述

〖摘要〗 有向概率图模型又称贝叶斯网络，

〖原文〗ccs228-notes

〖参考〗CMU 10-708 Slides / CMU 10-708 Lecture Notes / Jordan TextBook, Ch.2(section 2.1) / Koller’s Textbook,Ch. 3

1 简介

我们从 “表示” 的主题开始：如何选择概率分布来模拟世界的某些有趣的方面？

想出一个好的模型并不总是那么容易：在前面已经看到，一个简单的垃圾邮件分类模型需要我们指定一些参数，这些参数与英语单词的数量成指数关系！

在本章中，我们将学习一种避免这些症状的方法。我们准备：

  • 学习一种有效且通用的技术，仅使用几个参数来参数化概率分布。
  • 了解如何通过有向无环图 (DAG) 优雅地描述生成模型。
  • 研究 DAG 的结构与其所描述的分布以及建模假设之间的联系；这不仅会使这些建模假设更加明确，而且还将帮助我们设计更有效的推断算法。

本文中的各种模型都是有向图，也被称为『贝叶斯网络』。我们在后面还会看到另外一种方法：无向图，也称为马尔可夫随机场 (MRF)。 贝叶斯网络能够有效地展示因果关系，而马尔可夫随机场不能。因此，对于随机变量之间没有明确因果关系的问题，马尔可夫随机场更可取。

  • 有向图模型：有向边给出了随机变量之间的因果关系。
  • 无向图模型：无向边给出随机变量之间的相关性。

2 有向图模型

贝叶斯网络是一个有向无环图 (DAG)，其节点对应于随机变量，边对应于父变量对子变量的直接影响。贝叶斯网络能够以因子分解方式表示随机变量的联合概率，从而实现一组关于分布的条件独立假设的紧凑表示。贝叶斯网络中的每个变量（节点）都可以被视为其父项的随机函数。

有向图模型（贝叶斯网络）指一个概率分布族，该分布族允许紧凑的参数化，并可以使用有向图自然地进行描述。 这种参数化背后的总体思路非常简单。

回想一下，通过链式法则，我们可以将任何概率 ppp 写为：

p(x1,x2,…,xn)=p(x1)p(x2∣x1)⋯p(xn∣xn−1,…,x2,x1)。p(x_1, x_2, \dotsc, x_n) = p(x_1) p(x_2 \mid x_1) \cdots p(x_n \mid x_{n-1}, \dotsc, x_2, x_1)。 p(x1​,x2​,…,xn​)=p(x1​)p(x2​∣x1​)⋯p(xn​∣xn−1​,…,x2​,x1​)。

紧凑的贝叶斯网络是一个分布，其中右侧的每个因子仅取决于少数 祖先变量 xAix_{A_i}xAi​​：

p(xi∣xi−1,…,x1)=p(xi∣xAi)p(x_i \mid x_{i-1}, \dotsc, x_1) = p(x_i \mid x_{A_i}) p(xi​∣xi−1​,…,x1​)=p(xi​∣xAi​​)

例如，在一个有五个变量的模型中，我们可以选择用 p(x5∣x4,x3)p(x_5 \mid x_4, x_3)p(x5​∣x4​,x3​) 来近似因子 p(x5∣x4,x3,x2,x1)p(x_5 \mid x_4, x_3, x_2, x_1)p(x5​∣x4​,x3​,x2​,x1​)。在这种情况下，我们写成 xA5={x4,x3}x_{A_5} = \{x_4, x_3\}xA5​​={x4​,x3​}。

当变量是离散型时（现实问题中经常出现），我们可以将因子 p(xi∣xAi)p(x_i\mid x_{A_i})p(xi​∣xAi​​) 视为一个 概率表，其中行对应于 xAix_{A_i}xAi​​ 而列对应于 xix_ixi​ 的可选值；这些条目包含了实际概率 p(xi∣xAi)p(x_i\mid x_{A_i})p(xi​∣xAi​​)。如果每个变量可以取 ddd 个值并且最多有 kkk 个祖先，那么整个表将最多包含 O(dk+1)O(d^{k+1})O(dk+1) 个条目。由于每个变量都有一个表，因此整个概率分布只需要最多 O(ndk+1)O(nd^{k+1})O(ndk+1) 个参数来紧凑地描述（与朴素的 O(dn)O(d^n)O(dn) 相比）。

2.1 有向图表示

这种形式的分布可以自然地表示为 有向无环图（Directed Acyclic Graphics, DAG），其中顶点对应于随机变量 xix_ixi​，边表示随机变量之间的依赖关系。特别地，我们将每个节点 xix_ixi​ 的父节点设置为其祖先 xAix_{A_i}xAi​​。

例如，考虑一个学生在考试中的成绩 ggg 的模型。

该成绩取决于考试的难度 ddd 和学生智力 iii；同时它会影响该课程教师的推荐信质量 lll。学生智力 iii 也会影响 SAT 分数 sss。上述变量中，除了 ggg 可以取 3 个可能值，其他变量都是二值的。下图为该模型的有向概率图：

图中 5 个变量的联合概率分布自然地可以分解为：

p(l,g,i,d,s)=p(l∣g) p(g∣i,d) p(i) p(d) p(s∣i)p(l, g, i, d, s) = p(l \mid g)\, p(g \mid i, d)\, p(i)\, p(d)\, p(s \mid i) p(l,g,i,d,s)=p(l∣g)p(g∣i,d)p(i)p(d)p(s∣i)

此分布的概率图表示显然是一个有向无环图，它直观地指定随机变量之间的相互依赖关系。该图清楚地表明：推荐信取决于成绩等级，而成绩等级又取决于学生智力和考试难度。

另一种解释有向图的方法是描述数据的生成方式。在上例中，为了确定推荐信的质量，我们首先对智力水平和考试难度进行采样；得到这些参数后，对学生成绩等级进行采样；最后，根据该成绩生成推荐信。

回顾前面的垃圾邮件分类示例，我们隐含地假设了电子邮件是根据两步过程生成的：

  • 首先，我们选择垃圾邮件/非垃圾邮件标签 yyy；
  • 然后，我们以该标签为条件，独立地为每个单词采样是否存在。

2.2 形式定义

形式上，贝叶斯网络是一个有向图 G=(V,E)G = (V,E)G=(V,E)，该图中

  • 每个节点 i∈Vi \in Vi∈V 对应于随机变量 xix_ixi​。
  • 每个节点都有一个条件概率分布 (CPD) p(xi∣xAi)p(x_i \mid x_{A_i})p(xi​∣xAi​​)，指明了以 xix_ixi​ 的父节点取值为条件的节点概率。

因此，贝叶斯网络定义了一个概率分布 ppp。我们可以说，如果概率分布 ppp 能够被分解为一组由有向无环图 GGG 指定的因子乘积，则可以称概率 ppp 被 因子分解 在图 GGG 上。

不难看出，由贝叶斯网络表示的概率是有效的：显然，它将是非负的，并且可以使用归纳法（使用 CPD 是有效概率的事实）证明图中所有变量的所有可能赋值概率之和都为 “1”。相反，我们也可以通过反例证明，当 GGG 包含环路时，其相应的概率之和可能不等于 “1”。

3 贝叶斯网络的依赖关系

总而言之，贝叶斯网络表示了一个 “可以通过较小的局部条件概率分布（每个变量一个）的乘积形成” 的概率分布。通过这种表达概率的形式，我们能够在模型中引入一些变量独立性假设。

这同时也提出了一个问题：通过使用由指定结构的 GGG 描述的贝叶斯网络模型，我们究竟做出了哪些独立性假设？这个问题非常重要，有两个原因：

  • 我们应当也必须准确地知道正在做出何种模型假设（以及它们是否正确）
  • 这些信息将直接影响后面的统计推断任务，帮助我们设计更有效的推断算法。

让我们使用符号 I(p)I(p)I(p) 来表示在联合分布 ppp 中成立的所有独立性的集合。例如，如果 p(x,y)=p(x)p(y)p(x,y) = p(x) p(y)p(x,y)=p(x)p(y)，那么我们说 x⊥y∈I(p)x \perp y \in I(p)x⊥y∈I(p)。

3.1 有向图描述的独立性

事实证明，贝叶斯网络 ppp 非常优雅地描述了 I(p)I(p)I(p) 中的许多独立性；通过查看三种类型的结构，可以从贝叶斯网络中恢复这些独立性。

为简单起见，让我们首先查看具有 A，B，CA，B，CA，B，C 三个节点的贝叶斯网络 GGG。在这种情况下，GGG 本质上只有三种可能的结构，每一种都导致不同的独立性假设。感兴趣的读者可以使用一些代数轻松证明这些结果。

  • 共同父亲：如果 GGG 的形式为 A←B→CA \leftarrow B \rightarrow CA←B→C，并且观测到 BBB，则有 A⊥C∣BA \perp C \mid BA⊥C∣B 。但如果未观测到 BBB ，则有 A⊥̸CA \not\perp CA⊥C。直观地说，这源于 BBB 包含决定 AAA 和 CCC 结果的所有信息；一旦被观测到，就没有其他因素会影响这些变量的结果。

  • 级联：如果 GGG 的形式为 A→B→CA \rightarrow B \rightarrow CA→B→C，并且观测到 BBB，则 A⊥C∣BA \perp C \mid BA⊥C∣B。但是，如果 BBB 未被观测到，则 A⊥̸CA \not\perp CA⊥C。在这里，直觉再次是 BBB 持有决定 CCC 结果的所有信息；因此，AAA 取什么值并不重要。

  • V-结构（也称为 explaining away）：如果 GGG 的形式为 A→C←BA \rightarrow C \leftarrow BA→C←B，则知道 CCC 将耦合 AAA 和 BBB。换句话说，如果 CCC 未被观测到，则 A⊥BA \perp BA⊥B，但如果 CCC 被观测到，则 A⊥̸B∣CA \not\perp B \mid CA⊥B∣C。这种情况需要额外的解释。假设 CCC 是一个布尔变量，表示我们的草坪是否在一天早上潮湿； AAA 和 BBB 是它潮湿的两种解释：要么下雨（以 AAA 表示），要么洒水器打开（以 BBB 表示）。如果知道草是湿的（CCC 为真）并且洒水器没有开（BBB 为假），那么 AAA 为真的概率必须为 111，因为这是唯一可能的解释。因此，在给定 CCC 的情况下，AAA 和 BBB 之间并不是独立的。

上述三种结构清楚地描述了由三变量贝叶斯网络编码的各种独立性。我们可以在任何更大的概率图上递归地使用它们，以扩展到更一般的贝叶斯网络上。这导致了一个称为 ddd-分离（ddd-separation） 的新概念（其中 ddd 代表有向）。

假设 QQQ、WWW 和 OOO 是贝叶斯网络 GGG 中的三组节点。我们称 QQQ 和 WWW 之间在给定 OOO （即观测到 OOO）时是 ddd -分离的，如果 QQQ 和WWW 之间没有其他 活动路径 连接。

这涉及活跃路径的定义，GGG 中的无向路径在给定观测变量 OOO 的情况下被称为 活跃的，如果对于路径上每个连续的变量 X,Y,ZX,Y,ZX,Y,Z 三元组，以下其中一项成立：

  • X←Y←ZX \leftarrow Y \leftarrow ZX←Y←Z，并且 YYY 未被观测到 Y∉OY \not\in OY∈O
  • X→Y→ZX \rightarrow Y \rightarrow ZX→Y→Z，YYY 未被观测到 Y∉OY \not\in OY∈O
  • X←Y→ZX \leftarrow Y \rightarrow ZX←Y→Z，YYY 未被观测到 Y∉OY \not\in OY∈O
  • X→Y←ZX \rightarrow Y \leftarrow ZX→Y←Z，并观测到YYY 或其任何后代。

例如，在上图中，给定 X2、X3X_2、X_3X2​、X3​，X1X_1X1​ 和 X6X_6X6​ 是 ddd -分隔的。但是，给定 X1,X6X_1, X_6X1​,X6​ 时，X2,X3X_2, X_3X2​,X3​ 之间不是 ddd -分离的，因为可以找到一条活动路径 (X2,X6,X5,X3)(X_2, X_6, X_5, X_3)(X2​,X6​,X5​,X3​)

🪐 备注：

一位前 CS228 课程的学生创建了一个 交互式网络模拟网页 来测试 ddd-separation。

ddd -分离的概念很有用，因为它让我们能够描述模型中的大块依赖项。设 I(G)={(X⊥Y∣Z):X,Y are d-sep given Z}I(G) = \{(X \perp Y \mid Z) : \text{X,Y are d-sep given Z}\}I(G)={(X⊥Y∣Z):X,Y are d-sep given Z} 是 GGG 中一组 ddd -分离的变量。

事实 ：如果 ppp 对 GGG 上做因子分解，则 I(G)⊆I(p)I(G) \subseteq I(p)I(G)⊆I(p)。在这种情况下，我们说 GGG 是 ppp 的 III -映射（独立映射的意思）。

换句话说，在 GGG 中编码的所有独立性都是正确的：在 GGG 中 ddd 分隔的变量在 ppp 中也是真正独立的。然而，反之则不成立：一个分布可能会在 GGG 上因子分解，但可能具有 GGG 中未捕获的独立性。

在某种程度上，这几乎是一个微不足道的陈述。如果 p(x,y)=p(x)p(y)p(x,y) = p(x)p(y)p(x,y)=p(x)p(y)，那么这个分布仍然在图 y→xy \rightarrow xy→x 上进行因子分解，因为我们总是可以把它写成 p(x,y)=p(x∣y)p(y)p(x ,y) = p(x\mid y)p(y)p(x,y)=p(x∣y)p(y) 带一个 CPD p(x∣y)p(x\mid y)p(x∣y) ，其中 xxx 的概率实际上不随 yyy 变化。但是，我们可以通过简单地删除不必要的边来构造一个与 ppp 结构匹配的图。

3.2 有向图的表示能力

这提出了我们最后一个或许也是最重要的问题：有向图能否表达任何分布 ppp 的所有独立性？更正式地说，给定一个分布 ppp，我们可以构造一个图 GGG 使得 I(G)=I(p)I(G) = I(p)I(G)=I(p) 吗？

首先，注意很容易构造一个 GGG 使得 I(G)⊆I(p)I(G) \subseteq I(p)I(G)⊆I(p)。一个完全连接的有向无环图 GGG 对于任何分布都是一个 III-映射，因为 I(G)=∅I(G) = \emptysetI(G)=∅。

一个更有趣的问题是，我们是否可以为 ppp 找到一个最小的 III -映射 GGG，也就是说，找到一个 III -映射 GGG，使得即使从 GGG 中删除一条边也会导致它不再是 III-映射。这很简单：我们可以从一个完全连接的 GGG 开始并删除边，直到 GGG 不再是 III-映射为止。一种方法是遵循图的自然拓扑排序，并删除节点祖先，直到不再可能为止；在进行结构学习时，我们将在课程结束时重新审视这种修剪方法。

然而，我们真正感兴趣的是确定任何概率分布 ppp 是否总是承认一个 完美 映射 GGG，其中 I(p)=I(G)I(p)=I(G)I(p)=I(G)。不幸的是，答案是否定的。例如，考虑以下三个变量 X,Y,ZX,Y,ZX,Y,Z 上的分布 ppp：我们从伯努利分布中采样 X,Y∼Ber(0.5)X,Y \sim \text{Ber}(0.5)X,Y∼Ber(0.5)，并且设置 Z=X xor YZ = X \text{ xor } YZ=X xor Y（我们称之为 “含嘈异或” ）。可以使用一些代数 {X⊥Y,Z⊥Y,X⊥Z}∈I(p),but Z⊥{Y,X}∉I(p)\{X \perp Y, Z \perp Y, X \perp Z\} \in I(p), \text{but}\, Z \perp \{Y,X\} \not\in I(p){X⊥Y,Z⊥Y,X⊥Z}∈I(p),butZ⊥{Y,X}∈I(p) 来验证。因此，X→Z←YX \rightarrow Z \leftarrow YX→Z←Y 是 ppp 的 III -映射，但是我们之前讨论的 3 节点图结构都无法完美描述 I(p)I(p)I(p)，因此这个分布没有完美映射。

一个相关的问题是完美映射在存在时是否是唯一的。同样，情况并非如此，因为 X→YX \rightarrow YX→Y 和 X←YX \leftarrow YX←Y 编码相同的独立性，但会形成不同的图。更一般地说，如果两个贝叶斯网络 G1、G2G_1、G_2G1​、G2​ 编码相同的依赖关系 I(G1)=I(G2)I(G_1) = I(G_2)I(G1​)=I(G2​)，则它们是 III-等价的。
