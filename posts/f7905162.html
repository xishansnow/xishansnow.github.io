Linux зҺҜеўғй…ҚзҪ®ж–Үд»¶еӨ„зҗҶжөҒзЁӢ

д»ҠеӨ©йҒҮеҲ°дёҖдёӘй—®йўҳпјҢеҗҢж ·зҡ„иҪҜд»¶д»Һз»Ҳз«ҜиҝҗиЎҢиғҪеӨҹжӯЈеёёдҪҝз”Ёдёӯж–Үиҫ“е…Ҙжі•пјҢдҪҶд»ҺеӣҫеҪўз•ҢйқўдёӢиҝҗиЎҢе°ұж— жі•жӯЈеёёеӨ„зҗҶдёӯж–Үиҫ“е…Ҙжі•дәҶгҖӮжҖқиҖғдәҶдёӢпјҢж„ҹи§үеә”иҜҘжҳҜдёӨиҖ…зҡ„зҺҜеўғй…ҚзҪ®ж–Үд»¶дёҚеҗҢеҜјиҮҙзҡ„гҖӮжҹҘйҳ…дәҶдёӢ Linux зҡ„зӣёе…іж–ҮжЎЈпјҢзЎ®е®һеӯҳеңЁиҝҷдёӘй—®йўҳпјҢеңЁжӯӨи®°еҪ•еӨҮеҝҳгҖӮ

1 Shell зҺҜеўғй…ҚзҪ®ж–Үд»¶зҡ„еӨ„зҗҶжөҒзЁӢ

1.1 й…ҚзҪ®еҠ иҪҪжөҒзЁӢ

  • е…ЁеұҖй…ҚзҪ®ж–Үд»¶пјҡдҪҚдәҺ /etc/profile

  • з§Ғжңүй…ҚзҪ®ж–Үд»¶пјҡдҪҚдәҺ ~/.profile

  • Shell иҮӘе®ҡд№үй…ҚзҪ®ж–Үд»¶пјҢ дҫқжҚ®жҢҮе®ҡзҡ„ Shell жңүжүҖдёҚеҗҢпјҢдҫӢеҰӮпјҡ

    • еҜ№дәҺ bash пјҢжңү (~/.bash_profile | ~/.bash_login) -> ~/.bashrc -> /etc/bashrc
    • еҜ№дәҺ zshпјҢжңү ~/.zshrc

1.2 зҺҜеўғеҸҳйҮҸзҡ„и®ҫзҪ®

йҷӨдәҶеңЁдёҠиҝ°й…ҚзҪ®ж–Үд»¶дёӯпјҢйҖҡиҝҮ export еҸҳйҮҸеҗҚ=еҸҳйҮҸеҖј иҝҷз§Қж–№ејҸи®ҫзҪ®зҺҜеўғеҸҳйҮҸеӨ–пјҢLinux иҝҳеҸҜд»ҘйҖҡиҝҮеҚ•зӢ¬зҡ„й…ҚзҪ®ж–Үд»¶и®ҫзҪ®зҺҜеўғеҸҳйҮҸгҖӮиҝҷдәӣй…ҚзҪ®ж–Үд»¶зҡ„з»“жһ„д№ҹйқһеёёз®ҖеҚ•пјҢжҜҸиЎҢи®ҫзҪ®дёҖдёӘзҺҜеўғеҸҳйҮҸпјҢиҜӯжі•ж јејҸдёә еҸҳйҮҸеҗҚ=еҸҳйҮҸеҖјгҖӮе…¶дёӯпјҡ

  • е…ЁеұҖзҺҜеўғеҸҳйҮҸпјҡ дҪҚдәҺ /etc/environment дёӯ
  • з§ҒжңүзҺҜеўғеҸҳйҮҸпјҡ дҪҚдәҺ ~/.pam_environment дёӯ

1.3 еҲ°еә•дҪҝз”Ёе“ӘдёӘж–Үд»¶еҒҡй…ҚзҪ®пјҹ

  • еҰӮжһңеёҢжңӣзі»з»ҹзҡ„жүҖжңүз”ЁжҲ·йғҪе…·жңүеҗҢдёҖй…ҚзҪ®пјҢеҲҷдҪҝз”Ё /etc/profile е’Ң /etc/environment гҖӮ

  • еҰӮжһңеёҢжңӣжҹҗдёӘз”ЁжҲ·зҡ„жүҖжңү Shell е…·жңүеҗҢдёҖй…ҚзҪ®пјҢеҲҷдҪҝз”Ё ~/.profile е’Ң ~/.pam_environmentгҖӮ

  • еҰӮжһңд»…еёҢжңӣжҹҗдёӘз”ЁжҲ·зҡ„жҹҗдёӘзү№е®ҡ Shell е…·жңүзү№е®ҡй…ҚзҪ®пјҢеҲҷдҪҝз”Ё ~/.bashrc гҖҒ~/.zshrc зӯүгҖӮ

еҜ№дәҺеҸӘжңүдёҖдёӘз”ЁжҲ·зҡ„дёӘдәәз”өи„‘жЎҢйқўеә”з”ЁеңәжҷҜжқҘиҜҙпјҢйҖҡеёёеҸӘйңҖиҰҒж №жҚ®дҪҝз”Ёзҡ„ Shell еңЁ ~/.bashrc зӯүж–Үд»¶дёӯй…ҚзҪ®еҚіеҸҜпјҢеҰӮжһңйў‘з№ҒеңЁеӨҡдёӘ Shell д№Ӣй—ҙеҲҮжҚўпјҢеҲҷеҸҜд»ҘиҖғиҷ‘еңЁ ~/profile е’Ң ~/.pam_environment дёӯдёҖеҠіж°ёйҖёең°й…ҚзҪ®гҖӮ

2 X еӣҫеҪўз®ЎзҗҶеҷЁзҺҜеўғй…ҚзҪ®ж–Үд»¶зҡ„еӨ„зҗҶжөҒзЁӢ

2.1 дёҚеҗҢдәҺ Shell зҡ„зҺҜеўғй…ҚзҪ®

йҰ–е…ҲиҰҒзҗҶи§ЈпјҢLinux еӣҫеҪўз®ЎзҗҶеҷЁзҡ„зҺҜеўғеҸӮж•°дёҺ Shell дёӯзҺҜеўғеҸӮж•°жҳҜдёҚеҗҢзҡ„дёӨеҘ—дҪ“зі»гҖӮйҖҡеёёеңЁдёҖдәӣеӣҫеҪўз®ЎзҗҶеҷЁзҡ„еҗҜеҠЁиҝҮзЁӢдёӯпјҢдјҡе…ҲеҠ иҪҪ Shell зҺҜеўғй…ҚзҪ®ж–Үд»¶пјҲеҰӮпјҡ /etc/profileгҖҒ~/.profile зӯүпјүпјҢз„¶еҗҺеҶҚеҠ иҪҪеӣҫеҪўз®ЎзҗҶеҷЁзҡ„зҺҜеўғй…ҚзҪ®ж–Үд»¶пјҲеҰӮпјҡ /etc/xprofile гҖҒ /usr/local/etc/xprofile гҖҒ~/.xprofile зӯү пјүпјҢдҪҶдёәзЁіеҰҘиө·и§ҒпјҢжңҖеҘҪе°Ҷ Shell е’ҢеӣҫеҪўзҺҜеўғзҡ„й…ҚзҪ®еҲҶејҖи®ҫзҪ®гҖӮеҗҰеҲҷпјҢеҫҲе®№жҳ“еҮәзҺ°еңЁ Shell дёӯиҝҗиЎҢиүҜеҘҪпјҢиҖҢеңЁеӣҫеҪўз®ЎзҗҶеҷЁдёӯиҝҗиЎҢж—¶еҚҙејӮеёёзҡ„зҺ°иұЎпјҲжҲ–еҸҚд№ӢпјүгҖӮ

2.2 дё»иҰҒзҡ„зҺҜеўғй…ҚзҪ®ж–Үд»¶

ж¶үеҸҠеӣҫеҪўз•ҢйқўзҺҜеўғзҡ„й…ҚзҪ®ж–Үд»¶еёёи§Ғзҡ„жңүдёүз§Қпјҡ.Xsession гҖҒ.xinitrc е’Ң .xprofile гҖӮе…¶дёӯ .xinitrc жҳҜеҺҶеҸІжңҖжӮ д№…зҡ„й…ҚзҪ®ж–Үд»¶пјҢиҖҢ .Xsession еҲҷиў«еҫҲеӨҡзӨҫеҢәжҺЁиҚҗдёҺ .xinitrc дҝқжҢҒдёҖиҮҙпјҢдҫӢеҰӮпјҡдҪҝз”Ё ln -s .xinitrc .xsession е»әз«ӢиҪҜй“ҫжҺҘгҖӮдҪҶдёҚеҗҢзҡ„ Linux еҸ‘иЎҢзүҲе’ҢеӣҫеҪўз®ЎзҗҶеҷЁе…·дҪ“дҪҝз”Ёе…¶дёӯе“ӘдёҖдёӘжҲ–е“ӘдёҖдәӣпјҢеҫҖеҫҖйғҪжңүиҮӘе·ұзҡ„зәҰе®ҡгҖӮе®һйҷ…з»ҸйӘҢдёӯпјҢдҪҝз”Ё .xinitrc е’Ң .xprofile зҡ„жҜ”иҫғжҷ®йҒҚгҖӮдҫӢеҰӮпјҢеңЁеҺҹз”ҹжғ…еҶөдёӢпјҢ .xinitrc е’Ң .xprofile йҖҡеёёйғҪдјҡиў«д»ҘдёӢеӣҫеҪўз®ЎзҗҶеҷЁеј•з”Ёпјҡ

  • KDMпјҡеңЁ /usr/share/config/kdm/Xsession дёӯиў«еј•з”Ёпјӣ

  • LightDMпјҡеңЁ /etc/lightdm/Xsession дёӯиў«еј•з”Ёпјӣ

  • LXDMпјҡеңЁ /etc/lxdm/Xsession дёӯиў«еј•з”Ёпјӣ

  • SDDMпјҡеңЁ /usr/share/sddm/scripts/Xsession дёӯиў«еј•з”ЁгҖӮ

йӮЈд№ҲпјҢ.xprofile зӣёиҫғдәҺ .xinitrc/.xsession жңүд»Җд№Ҳе·®еҲҘе‘ў?

  • .xprofile жҳҜиў«еӣҫеҪўз®ЎзҗҶеҷЁпјҲ GDM/KDMзӯүпјүи°ғз”Ёзҡ„пјҢжүҖд»Ҙе®ғжҜ”иҫғеҚ•зәҜпјҢеҸӘйңҖиҰҒеңЁйҮҢйқўеҠ иҪҪдҪ зҡ„зҺҜеўғй…ҚзҪ®е’Ңе‘Ҫд»ӨеҚіеҸҜпјҢ .xprofile иў«жү§иЎҢзөҗжқҹд№ӢеҗҺпјҢжҺ§еҲ¶жқғдјҡиҝ”еӣһеӣҫеҪўз®ЎзҗҶеҷЁгҖӮ

  • .xinitrc/.xsession жҜ”иҫғеӨҚжқӮдёҖдәӣпјҢеӣ дёә .xinitrc жҲ– .xsession ж–Үд»¶зҡ„жңҖеҗҺдёҖеҸҘеә”еҪ“жҳҜ exec gnome-session жҲ– exec startkde жҲ– exec enlightenment жҲ– exec icewm д№Ӣзұ»зҡ„дјҡиҜқзЁӢеәҸпјҢж— и®әе“Әз§ҚеӣҫеҪўз®ЎзҗҶеҷЁпјҢеңЁи°ғз”ЁиҝҷдёӨдёӘж–Үд»¶ж—¶пјҢйҖҡеёёйғҪж„Ҹе‘ізқҖжҺ§еҲ¶жқғдјҡиў«зӣёеә”зҡ„зӘ—еҸЈз®ЎзҗҶеҷЁжҺҘз®Ў гҖӮ

2.3 й…ҚзҪ®ж–Үд»¶еҠ иҪҪиҝҮзЁӢ

еҜ№дәҺеӨ§йғЁеҲҶеә”з”ЁеңәжҷҜжқҘиҜҙпјҢжҲ‘д»¬еҸӘйңҖиҰҒй…ҚзҪ® .xprofile еҚіеҸҜгҖӮдёӢйқўжҳҜжҹҗеӣҫеҪўз®ЎзҗҶеҷЁеҗҜеҠЁж—¶пјҢи°ғз”ЁзҺҜеўғй…ҚзҪ®ж–Үд»¶зҡ„иҝҮзЁӢгҖӮ

2.4 еҲ°еә•иҜҘз”Ёе“ӘдёӘй…ҚзҪ®ж–Үд»¶пјҹ

  • еҰӮжһңеёҢжңӣзі»з»ҹдёӯжүҖжңүз”ЁжҲ·зҡ„еӣҫеҪўз•ҢйқўпјҢйғҪдә«жңүеҗҢдёҖй…ҚзҪ®пјҢдҪҝз”Ё /etc/xprofile жҲ– /usr/local/etc/xprofile

  • еҰӮжһңеёҢжңӣжҹҗдёӘз”ЁжҲ·зҡ„еӣҫеҪўз•ҢйқўжңүиҮӘе·ұзү№е®ҡзҡ„й…ҚзҪ®пјҢдҪҝз”Ё ~/.xprofile
